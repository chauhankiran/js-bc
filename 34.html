Syntactical Sugar

In order to iterate through an array, we now have three possible solutions. As we moved from for to .forEach() to for...of readability of code increase.

These three options are there for you to just ease of programming. This is what we called syntactical sugar on syntax to make syntax or code looks "sweet". At the end, behind the scene same basic operation is running at bottom level of JavaScript but with this sugar added syntax, it now look nicer.
