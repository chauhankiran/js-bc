.forEach

This function works on array e.g. you need to call on array variable such as: 

a = [34, 45, 47, 40, 33]

a.forEach()

This forEach() function accept function as parameter such as: 

a = [34, 45, 47, 40, 33]

a.forEach(function() {

})

This .forEach() function gives you each element from an array a (again element not index) which needs to pass as parameter of function we just defined: 

a = [34, 45, 47, 40, 33]

a.forEach(function(i) {

})

Now, you can do whatever you want with this i such as, 

a = [34, 45, 47, 40, 33]

a.forEach(function(i) {
  console.log(i)
})

See, much cleaner syntax!
