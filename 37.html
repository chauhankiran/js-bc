JSON

While talking about JavaScript objects let not forget the JSON. We're going to use this a lot and I'll talk about JSON a lot in future sessions. JSON is similar to JavaScript object except

  1. must be name value paired,
  2. name must be string and enclosed within double quotes only,
  3. separated by comma but not allowed hanging comma.

Putting this in action - 

let name = {
  "first_name": "Bruce",
  "last_name": "Wayne",
  "is_supernatural": false
}
