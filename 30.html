Iteration on Array

One of the popular(!) and easy way to do the iteration or loop through the array is using for loop, where you run the loop from 0 to length of array. 

a = [34, 45, 47, 40, 33]

for (let i = 0; i < a.length; i++) {
  console.log(a[i])
}

But, you'll not see this way to iterate an array except in old code. Because, you now have nicer way to loop through the array such as usage of .forEach() function.
