for...of Iteration

If you don't like .forEach() function, you can use for...of construct. 

marks = [34, 45, 47, 40, 33]

for () {

}

Instead of passing that old three arguments separated by comma with for, you need to pass a variable, of and then an array such as: 

marks = [34, 45, 47, 40, 33]

for (let mark of marks) {
  console.log(mark)
}

This will also give the identical result.
