Array Destructuring with Spread Syntax

Consider the following example: 

let marks = [34, 35, 37, 45, 40, 56]

These are the marks of seven subjects. I'm interested in first, second and then remaining. In other words, I want to store 34 in one variable, 35 in second variable and remaining elements in third variable as array.

With destructuring and spread operator, it is trivial: 

let [math, physics, ...other_marks] = marks

With this, math variable hold 34, physics variable hold 35, and other_marks - an array variable hold remaining marks - [37, 45, 40, 56].

Destructuring and spread operator used a lot in React.
