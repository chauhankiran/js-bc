Useful Constants

We'll solve the error in type.js within few minutes. But, while we're here, I want to point out two important constants that'll use a lot on back-end side. These constants are - __dirname and __filename.

__dirname return the current directory name and __filename return current file name in execution. Let's create new program with name const.js and print the output from both constants as follows: 

console.log(__direname)
console.log(__filename)

Running this program will print the following output: 

$ node const.js
/home/kiran/Code/Node
/home/kiran/Code/Node/const.js

The first one print the directory or folder name and second one print the name of file.
