os.type()

While we're here, why not try out other object(s)? How about os.type()? If you look over the docs for os, you'll find a method with name type() and it return a simple string based on your operating system. As it returns, we need to a console.log. Let's create a type.js program and type following code: 

console.log(os.type())

Let's run this program: 

$ node type.js
/home/kiran/Code/Node/type.js:1
console.log(os.type())
            ^

We called pid constant from process object and it also work. But, now Node is complaining about os, even though it is written there in documentation.
