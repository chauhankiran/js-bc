Hello, World

Programming journey normally start with famous hello, world program and learning Node is not exception. Let's create hello.js file and write the following code: 

console.log("hello, world");

Nothing changed in Node. The extension is same, the code is same! But, they way we now execute this code is change. Instead of including this script file within HTML, you can directly run from terminal using following command: 

$ node hello.js
hello, world

node and then name of the file we want to execute!
