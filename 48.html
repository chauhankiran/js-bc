Array Destructuring

Instead of creating separate variable and assigning value to each just write - 

let [maths, physics, chemistry] = marks

JavaScript automatically assign first element of marks to maths, second element of marks to physics, and third element of marks to chemistry.

What if I don't want physics marks. Just leave the space off - 

let [maths, , chemistry] = marks

What if I only want chemistry marks. Just leave that you don't want - 

let [, , chemistry] = marks

Regarding above syntax, don't do this - this is not readable. You need to count comma, and the element to know what chemistry will get from array. In this case, I would suggest you to go with old way.
