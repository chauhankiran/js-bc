Suggestion

Although it is highly recommended that we should use same name as string id while doing require it is not compulsory. You can write following and it just work fine: 

const something_important = require('os')

something_important.type()

But, this program is not readable. Assume that somebody written 200 lines of code and somewhere around 145 line, you find important_stuff_from_linux object, how do you know what it is? Instead if os is written, you might get rough idea on that it is the os module from Node.
