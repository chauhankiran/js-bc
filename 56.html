V8

We all are familiar with web browsers such as Mozilla Firefox, Google Chrome and Apple Safari. They display webpages written in HTML, styles in CSS, and customize in JavaScript. This is possible because these browsers know these HTML, CSS, and JavaScript languages. When you install the web browser on your computer, you're installing compiler that compiles HTML, another compiler that compiles CSS, and yet another compiler that compiles JavaScript. Instead of compiler, they call it engine means web browsers have engine for HTML, CSS, and JavaScript. You might heard names such as Gecko, Blink or WebKit. These are the engine within browser that render the content. There is separate engine for JavaScript compilation called SpiderMonkey in Firefox, V8 in Chrome, and JavaScriptCore in Safari. They are the name of engines which execute JavaScript. Let's focus on V8 engine. It is JavaScript engine written for Chrome (Chromium) browser.
