Switch

You can write the previous condition code in switch as follows: 

let day = 0

switch (day) {
  case 0:
    // It's Monday.
    break;
  case 1:
    // It's Tuesday.
    break;
  case 2:
    // It's Wednesday.
    break;
  case 3:
    // It's Thursday.
    break;
  case 4:
    // It's Friday.
    break;
}

I'm not going into the details of break, default and how the execution flows. I assume that you already know this or you can read online as these concepts are too easy to understand by write few example code.
