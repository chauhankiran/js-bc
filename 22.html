How to Define Loop?

Using for keyword such as: 

let count = 5

for (let i = 0; i < count; i++) {
  // do something with i.
}

Or you might use while keyword such as: 

let count = 5

while (count < 5) {
  // do something with count.
  // count++
}

In extreme cases, you might use do...while loop.

Things are not just ends here, you also have programming constructs such as for...in or for...of and .forEach() method. We're going to use these three in future sessions.
