String Interpolation

Check out this example: 

first_name = "Bruce"
last_name = "Wayne"

console.log("Hello " + first_name + " " + last_name ". How are you?")

This example print the greeting message using string concatenation. I don't know about you, but for me its is like normal programming error that I miss something from above concatenation - it might be space or + or something else. But, I couldn't able to write string concatenation at the first try, it always send arrow to me!

Luckily, ES6 solved this with something called String Literals or String Interpolation where you just write a complete string or text you want to print including variables such as: 

first_name = "Bruce"
last_name = "Wayne"

console.log(`Hello ${first_name} ${last_name}. How are you?`)

While doing string interpolation, you need to use back-tick instead of single or double quote and then wrap your variable within ${} to differentiate it from normal text. No more + and calculation of spaces within string concatenation.
