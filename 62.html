Obvious Question

Now you might have question like which one we need to follow? My personal suggestion is:

  1. Always give .js extension to your file.
  2. Run the file with extension even though it is options
  3. Use folder and index.js based on requirement and environment you are working.

The reason why I mentioned all these because you'll see all these (except first one, which is rare but still used in some old project) used many different places.
