Function is First-Class Citizen

Function is a first-class citizen in JavaScript e.g. you can treat function as variable. You can even assign the complete function as value to variable such as: 

let add = function(a, b) {
  return a + b;
}

If you're assigning the function to a variable, you can eliminate the function keyword by using arrow or => after parameters and before opening curly brace such as: 

let add = (a, b) => {
  return a + b;
}

This way of defining a function is called arrow function.
