Suggestions

Although, I've used nonsensical name of variables to define arrays but you must avoid it while writing in projects or in bigger programs. The recommendation for naming array variable is -

  1. Name the variable after what it contains. For example, in above case, if the array contains marks out of 50, then we should give the name of array after mark or marks.
  2. Array name must be plural e.g. marks not mark. In this way, when you loop through it, you get a single element and can assign singular name of your array such as mark of marks.
  3. Also, while defining an array, put a space after comma in box brackets that make array elements more readable*.

Putting above suggestion into action our above program becomes: 

marks = [34, 45, 47, 40, 33]

marks.forEach(function(mark){
  console.log(mark)
})

While running a loop on marks, with each iteration, you'll get mark.
