Function with Default Parameters

Copying the previous define function here. Pay attention to its parameters. 

let add = (a, b) => {
  return a + b
}

In order to call this function, we need to write something like: 

add(23, 46)

But, with ES6, you can also define the default parameter values such as: 

let add = (a=23, b=46) => {
  return a + b
}

Now, we can call this function as: 

add()
add(34, 7)

The example is trivial and does not justify how useful this feature is, but in future session, we'll use this feature for default behavior.
