Function with Default Parameters

Copying the previous function: 

let add = (a, b) => {
  return a + b
}

In order to call this function, we need to write something like: 

add(23, 46)

But, with ES6, you can also define the default parameter values such as: 

let add = (a=23, b=46) {
  return a + b
}

Now, we can call this function 

add()
add(34, 7)

The example is trivial and does not justify how useful this feature, but in future we'll use this feature for default behavior.
