An Interactive Environment

Remember, the console from where you type JavaScript and execute instantly? You have the same interactive environment in Node. Just type node in terminal as follows: 

$ node
Welcome to Node.js v14.3.0.
Type ".help" for more information.
>

Now you can type the JavaScript and it'll going to execute after pressing Enter. For example, 

> console.log("hello, world")
hello, world
undefined
>

All the JavaScript code we write or see so far in first and second sub-sessions are valid in Node as well. As I mentioned previously, Node is a JavaScript runtime with more or less functionalities.

FYI: You can exit the environment using .exit command.
