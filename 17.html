Types

Even though JavaScript is dynamic typed programming language, but still it has some types such as:

  1. Numbers
  2. String
  3. Boolean
  4. Object
  5. Symbol
  6. undefined
  7. null

Arrays and functions are special types of objects:

  1. Numbers
  2. String
  3. Boolean
  4. Object
    1. Arrays
    2. Functions
  5. Symbol
  6. undefined
  7. null

Note: Totally okay, if you don't know about Symbols. It is new and we're going to cover this in future session.
