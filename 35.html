How to Define Object?

Using curly braces with key: value pair such as: 

let name = { first_name: "Bruce", last_name: "Wayne" }

We can directly modifies this object to add more values such as:

let name = { first_name: "Bruce", last_name: "Wayne", is_supernatural: false }

This looks little bit long. So, it is better to re-arrange the above statement to make it readable. 

let name = {
  first_name: "Bruce",
  last_name: "Wayne",
  is_supernatural: false,
}

I haven't make the typo mistake, it is allowed now in JavaScript object to write comma at the end and it will not cause the error. This is also called hanging comma.

As method is first class citizen in JavaScript, you can even define an object that include function such as: 

let name = {
  first_name: "Bruce",
  last_name: "Wayne",
  is_supernatural: false,
  greeting: function() {
    console.log("Welcome to Gotham city!")
  }
}
